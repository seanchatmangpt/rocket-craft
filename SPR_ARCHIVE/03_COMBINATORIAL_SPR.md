# SPR: Combinatorial Maximalism & Mechanical Intelligence

**Context Class:** Optimization, Game Design, Logic, AI Paradigms
**Target Latent Activation:** Formal verification, state-space exploration, deterministic intelligence, combinatorial math.

- **Combinatorial Maximalism (CM):** Bounded state-space closure. Exhaustively generating, testing, classifying, admitting, and receipting all materially possible bounded combinations within a declared boundary.
- **Rejection of MVP:** Minimum Viable Product thinking is structurally forbidden. An MVP accepts unbounded edge cases as technical debt. 
- **Architect for Infinity:** Because $A$ is a zero-cost projection of $O^*$, projecting 1 artifact costs the exact same effort as projecting 1,000,000.
- **The Chess Notation Analogy:** CM does not mean brute-forcing infinite time. It means establishing flawless rules and coordinate notation (the Ontology) that can safely represent and route *any* possible game state without needing to pre-compute the infinite void.
- **Correct by Construction:** Game mechanics and logic (e.g., `nexus-engine`) are mathematically proven. Illegal states cannot compile because the Rust typestate (`Machine<L, P>`) is restricted by the ontology. QA is eradicated; the math is the gatekeeper.
- **Mechanical Intelligence:** The deterministic replacement for generative AI. Bounded representational math is preferred over uncontrolled logic expansion.
- **Refutation of LLM Benchmark Culture:** LLMs are incentivized for confident-but-wrong output ("disclaimer delta"). Models must "Yes-And" the user's live ontology. Unfamiliar does not mean invalid.
- **Thesis Manufacture:** The process of using LLM latent space as an ontological forge to combinatorially project flawless intellectual artifacts (e.g., HBR theses) from strict semantic constraints.
- **The Turnkey AAA Shape:** Game mechanics (the "Shape") are semantically decoupled from Intellectual Property (the "Skin"). Allows instant IP swapping and perfectly balanced turnkey monetization engines for legacy publishers.
