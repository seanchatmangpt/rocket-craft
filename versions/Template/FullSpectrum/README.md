# FullSpectrum (Crafting Template)

> **Note:** This project is built and managed using the `./rocket` CLI tool.

## Purpose
A specialized template project titled **Crafting template**, part of the **Rocket-Craft** ecosystem. It serves as a foundational scaffold for implementing complex crafting mechanics and inventory systems.

## Target Platforms
- **Mobile**: Targeted hardware class for broad accessibility.
- **Standard PC**: Supports general UE4 development platforms.

## Unique Config Settings
- **Game Instance**: Utilizes a custom Rocket-Craft specific game instance: `/Game/Config-Rocket-Craft/main-config.main-config_C`.
- **Hardware Targeting**: Mobile with Scalable performance settings.
- **Rendering**: Mobile HDR is disabled to ensure consistent performance across target devices.
- **Maps**: 
  - Main Application: `Application`
- **General**: Minimal configuration setup, designed to be extended by specific crafting implementations.
