## Unlock Method Overview

| Unlock Type | Suits Available |
|-------------|-----------------|
| Gacha only | All SSR suits (on specific banners) |
| Campaign (free) | RX-78-2, Strike Gundam, Barbatos (Base) |
| AR Bridge (budget) | RX-78-2 (HG $12), Barbatos (HG $14), Strike ($15), Tallgeese (HG $15) |
| AR Bridge (premium) | All MG/PG kits ($35–$45 range) |
| Tournament reward | Wing Zero (#1 AC season), Epyon (top 8 AC melee) |
| Collector's exclusive | RX-78-2 Gold Version (PG kit only; no gacha path) |

## Parry Type Reference

| Parry Type | Suits | Notes |
|------------|-------|-------|
| Standard | RX-78-2, Strike, Justice, Tallgeese, Turn A | Widest broad windows; no system bonus |
| NT-Enhanced | Nu Gundam, Zeta, Sazabi, Unicorn, F91, Aerial | Psychoframe or equivalent; better perfect windows |
| AVJ-Enhanced | Barbatos Lupus Rex, Barbatos (Base) | Body-reading via spinal AVJ; no HP cost on Lupus Rex |
| ZERO-Assisted | Wing Zero, Gundam Epyon | Predictive system; auto-counter on perfect |
