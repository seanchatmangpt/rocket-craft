# RX-78-2 Gundam — Gold Version (Limited Collector's Edition)

**Series:** Mobile Suit Gundam (Universal Century 0079) — Collector's Variant
**Gacha Tier:** SSR (Limited)

## Lore
The Gold Version of the RX-78-2 is not a canonical mobile suit from the Gundam universe — it is the rarest object in the Gundam Nexus collectible ecosystem. Produced as an extremely limited physical model kit with genuine gold-plated parts and a unique registration code, the Gold Version carries the same lore as the original RX-78-2 but with an additional layer: it represents the pinnacle of what human craftsmanship and reverence for the original can achieve. In the game's lore, it is described as a Newtype vision — what the RX-78-2 might have been if Amuro had been given unlimited resources and time.

## Unique Mechanic: Gold Legacy & Collector's Aura
- Gold Legacy: Unlike the base RX-78-2's unlimited upgrade potential, Gold Version starts with all upgrades pre-applied at maximum efficiency
  - All base stats begin at their maximum upgraded values (no grinding required)
  - Each upgrade item applied BEYOND maximum grants a cosmetic "gilded" effect (visual enhancement, no stat gain) — lore-accurate to the "perfected" nature of this suit
- Collector's Aura: Passive — Gold Version has a permanent golden glow visible in all combat scenes
  - Enemy AI prioritizes attacking Gold Version last (awe-mechanic; only applies to story AI, not PvP)
  - Allies in party with Gold Version gain +5 ATK (inspirational presence)
- Historical Archive Complete: Gold Version begins with all 3 One Year War Archive bonuses already unlocked (+15 to stat allocation; distributed as +5 ATK, +5 DEF, +5 RES)
- Amuro's Final Echo: Once per match — when any ally is reduced to 0 HP, Gold Version performs a free counter-attack on the attacker (1.5× ATK; reflects Amuro's drive to protect his comrades)
- Prestige Parry: Perfect parries from Gold Version generate a golden shockwave (+2 damage to adjacent enemies; cosmetic and functional simultaneously)

## Signature Move: The Last Shooting (Gold Edition)
- Gesture: Hold 2.0s → Swipe Down → Swipe Up → Draw Circle → Release
- Effect: Gold Edition version of Last Shooting; beam rifle fires in golden beam with 3.5× ATK damage (vs. base's 2.8×); guaranteed stagger; if enemy HP below 40%, dual beam saber follow-up deals additional 2.0× ATK; leaves golden crater visual
- Cooldown: 120 seconds (reduced from base's 140s — "perfected timing")
- Visual: Gold Fin Funnel-inspired particle trails spiral around beam; entire screen momentarily gold

## Combo Chain
Perfected IB4 chain (1.2×/1.8×/2.5×/4.0× — starts at 1.2× instead of 1.0×, reflecting the pre-maximized state). Prestige Parry shockwave adds +2 to each combo step when triggered. Legacy Core upgrade stacks carry over from any base RX-78-2 save data (cross-suit inheritance, unique to Gold Version).

## Parry Type
Standard (Perfected): 235ms broad / 85ms perfect (widest windows in roster — the "final form" of Amuro's foundational technique; Prestige Parry visual triggers on every perfect)

## Striker Pack Compatibility
G-Fighter Pack (Gold Edition) — exclusive gold-plated variant; ATK +12 (vs. standard G-Fighter's +10); visual effect: golden flight trails; available only through additional collector's registration

## AR Bridge Integration
- PG RX-78-2 Gundam Gold Version (1/60): The primary unlock — registers as Gold Version SSR (Limited) (this profile). Physical kit MSRP $11,000; includes: holographic authentication card, numbered certificate, gold-plated runner set, special edition insert code
- No alternative AR unlock exists — the Gold Version is exclusively tied to its physical counterpart
- If kit is scanned without Digital Edition premium, unlocks only the Gold SR visual skin for base RX-78-2 (not SSR)

## Unlock Methods
- AR Bridge ONLY: PG RX-78-2 Gold Version physical kit (primary and only intended unlock)
- No Gacha: Gold Version does not appear on any gacha banner — it cannot be pulled
- No Campaign: Not available through story progression
- Emergency Protocol: Players who can verify auction purchase of the physical kit (receipt + serial number) may contact support for a one-time SSR unlock code (support discretion; not guaranteed)
